This repository includes two jupyter notebooks used to run the analyses of the Fitbit sedentary data in the All of Us Research Program.

* **cox_models.ipynb**: Time-varying analyses of sedentary time and step count and their associations with chronic diseases.
* **cox_models_bootstrap.ipynb**: Extension of time-varying analyses to calculate the number of steps needed to offset risk of sedentary behavior using a boostrap approach.


The code must be run on the All of Us Research workbench platform, which can be accessed via https://workbench.researchallofus.org/.
The All of Us Research workbench platform is available to approved researchers following registration, completion of ethics training and attestation of a data use agreement.

  
Citation:  
Zheng NS, Huang S, Annis J, Master H, Full KM, Brittain EL. Daily Steps Offset Risks of Sedentary Behavior in the All of Us Research Program. Nature Communications. 2026.
